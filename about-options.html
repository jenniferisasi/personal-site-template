Quick Decision Guide

Choose Option A if you:

  • Are pursuing a PhD or graduate degree
  • Want to show your academic progress visually
  • Need to track milestones and completion status
  • Want a more feature-rich landing page
  • Don't mind editing HTML (it's well-commented)

Choose Option B if you:

  • Prefer a simple, text-focused page
  • Want to use only Markdown (easier)
  • Don't need a timeline or progress tracking
  • Want minimal, clean design
  • Are new to web development
